# 125. Valid Palindrome

## 1️⃣ 題目描述
- 給定一個字串 `s`，判斷其是否為**有效回文（valid palindrome）**
- 有效回文定義：
  - 忽略大小寫
  - 忽略非英數字元（只保留字母與數字）
  - 經處理後字串需正著與反著都一樣

## 2️⃣ 解法思路

### 方法 1️⃣：清洗字串 + 反轉比較（O(n) 空間）
- **思路**：
  1. 遍歷整個字串，只保留英數字元，並統一轉成小寫
  2. 構造新字串 `filtered`
  3. 比較 `filtered` 與其反轉字串是否相同
- **適用情況**：邏輯直觀、容易實作，但需額外空間
